\begin{abstract}
\setcounter{page}{6}
\thispagestyle{plain}
The Finite Difference Time--Domain\index{FDTD} method is a differential numerical technique for solving electromagnetic wave scattering problems. In order to model metamaterials\index{metamaterial} with negative refractive index the standard algorithm is modified as implementation of the Drude\index{dispersive model!Drude} dispersion model. Suitable values of plasma frequency ($\omega_p$) are chosen to model the metamaterial cloaking\index{cloak} structure following the treatment of~\cite{Radial-Zhao}. The lossless case of electromagnetic cloak is implemented using a graphics processing unit\index{GPU} (GPU). Simulation performance on Matlab\index{Matlab}, C++\index{C++} and GPU\index{GPU} is analysed.
\end{abstract}
